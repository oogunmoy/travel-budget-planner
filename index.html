❓ How to Use

🚀 Getting Started

  • Set your Annual Travel Budget at the top of the dashboard
  • Select a year from the dropdown (or add a new one)
  • Click + Add Trip to create your first trip

✈️ Adding Trips & Stops

  • Each trip has a name, starting location, dates, and status (Planned / Booked / Completed)
  • Click + Add Stop inside a trip to add destinations
  • Each stop has its own expenses, dates, currency, and notes
  • Reorder stops by dragging them or using the ▲/▼ buttons
  • Shared expenses (like a round-trip flight) go in the "Shared / Trip-Wide" section

🧮 Entering Expenses

  • Type a number in any amount field and move on
  • Math expressions: type 10 + 7.50 + 15 and it calculates to $33 when you tab away
  • Click back into a field to see and edit the original formula
  • Works with + - * / (e.g., 150 * 3 for 3-night rates)

💱 Currency Converter

  • Set a stop's Currency (e.g., EUR) and Rate (e.g., 0.92 = 1 USD buys 0.92 EUR)
  • A column appears showing local currency amounts alongside USD
  • All amounts are stored in USD — local amounts are display-only

💳 Points, Credits & Budget

  • Points Used ($): dollar value of points/miles used (e.g., AMEX points covering a flight)
  • Credits ($): refunds, shared bill reimbursements, etc.
  • Out-of-Pocket = Total Spent − Points − Credits
  • Remaining Budget = Budget − Out-of-Pocket (points/credits give you budget back)

💾 Saving Your Data

  • Data auto-saves to your browser (localStorage) on every edit
  • Click 💾 Save File to bake data into the HTML file itself
  • The saved file is portable — move it to OneDrive, email it, open on another device
  • If you close without saving, the browser will warn you
💡 Think of it like a Word doc — you edit, you save, the file has your work. Always click 💾 Save File before moving the file somewhere else.

📊 Exporting to Excel

  • Click 📊 Export to Excel to download a workbook
  • One sheet per year, categories as columns, trips as rows
  • Stops are grouped under trips — expand/collapse with Excel's +/− buttons

📥 Importing from a Spreadsheet

  • Click 📥 Import and select an .xlsx file
  • The tool maps columns like "Departure Date", "From", "To", "Lodging", "Food", etc.
  • Each row becomes a trip — review and consolidate multi-stop trips after import

🔧 Other Features

  • 🔍 Search: type in the search bar to filter trips by name, destination, or notes
  • 📋 Duplicate: copy a trip's structure with zeroed amounts as a template
  • 📈 Year-over-Year: compare spending across all years (below the dashboard)
  • ☀️/🌙 Theme: toggle dark/light mode in the header
  • 🖨️ Print: use Ctrl+P — the layout auto-adjusts for paper
  • 🗑️ Clear Year: wipe all data for the selected year

🔒 Privacy

Your data never leaves your device. No servers, no accounts, no tracking. The only external request is loading the ExcelJS library for spreadsheet features.
